## 🤝 How to Contribute

1. Upload your resources to Google Drive and set sharing to "Anyone with the link can view".
2. Reply in this PR / open a new PR with the subject folder README updated in the format below:

```markdown
- [📘 Resource Title](https://drive.google.com/your-link)
  **Contributor:** YourGitHubUsername
```

3. Avoid committing large PDFs to the repository; prefer Drive links as per CONTRIBUTING.md.
